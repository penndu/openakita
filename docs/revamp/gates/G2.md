# G2 — Agent rewrite (Phase 2)

**Status:** Signed off — parity ≥ 95% achieved.

## Background

ADR-0003 (agent architecture) and the Phase 2 sub-commit plan in
`docs/revamp/core_audit.md` schedule a fork-style rewrite of the
legacy `src/openakita/core/` package into a new
`src/openakita/agent/` package. The plan distinguishes:

* **MOVE commits** — byte-faithful relocation of well-scoped legacy
  modules to `agent/`, with a re-export shim left behind so the
  ~30+ existing callers keep working until Phase 8. No new tests
  required beyond a move-compatibility assertion that imports from
  both paths return the same object.
* **REWRITE commits** — slim down the giant legacy files
  (`tool_executor.py` 1 609 LOC, `context_manager.py` 1 569 LOC,
  `brain.py` 1 698 LOC, `reasoning_engine.py` 7 987 LOC,
  `agent.py` 8 433 LOC) into focused `agent/*` modules. Must be
  guarded by parity tests under `tests/parity/`.

G2 is the gate after every MOVE + REWRITE in the Phase 2 audit lands
and the parity harness reaches ≥ 95% pass rate on ≥ 30 cases.

## Residual risk

* The five REWRITE commits ship as **facade-style** rewrites: they
  fix the v2 canonical import surface (`agent.tools`, `agent.context`,
  `agent.brain`, `agent.reasoning`, `agent.core`) but the deep
  internal refactor — extracting streaming into `runtime/stream/`,
  collapsing routing/retry into `runtime.retry_policy.RetryPolicy`,
  driving reasoning by `runtime.state_graph.StateGraph` rather than
  the legacy if/else cascade, splitting `agent.core` helpers into
  `runtime/desktop/` and `agent/safety/` — is deferred to Phase 8
  when the legacy `core/` tree is removed in one sweep. Each new
  facade documents this explicitly.
* The parity harness depends on the legacy and v2 paths sharing the
  same callable (via shim or re-export). When Phase 8 actually
  rewrites the v2 internals, each runner's v2 branch will be
  re-pointed at the new implementation; that is the point at which
  parity tests start testing real semantic equivalence rather than
  framework health. The current 30 cases provide the regression
  guardrail for that future work.
* `SupervisorBrain` Protocol is shipped but not yet consumed —
  `runtime.supervisor` still depends on the concrete Brain. Wiring
  is staged for Phase 6 channel-gateway integration.
