# AI 分镜师 / Storyboard

把一段文字想法拆成可拍可生成的镜头表。每镜头都有画面、镜头语言、时长、台词、声音提示。

## 给小白用户

1. 在文本框里写一段故事或想法
2. 设置目标时长（默认 30 秒）
3. 点【生成分镜】
4. 一键导出 CSV，可直接给摄影或 AI 生图用

## 三大特点

- **5 级 fallback 解析** — LLM 输出再乱也能解析出来
- **三分自检** — 自动检查时长 / 分布 / 镜头数是否合理
- **零依赖** — 没配 LLM 也能跑通（退化成等长切分）
